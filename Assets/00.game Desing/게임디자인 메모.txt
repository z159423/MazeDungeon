클래스 스킬
1. 모험가

2. 도적
- 은신(패시브) : 어둠속으로 은신합니다. 은신중 공격하면 은신이 풀립니다. 은신중 공격당하면 은신이 풀립니다. 은신중에는 이동속도가 줄어듭니다.
- 급습 : 적을 뒤에서 공격하면 추가대미지를 입힙니다. 은신중에 공격하면 더 큰 데미지를 입힙니다.
- 독찌르기 : 독을묻힌 단검을 찔러 적을 중독시킴
- 블레이드 댄스 : 전방을 향해 여러번 공격을 합니다.
- 대거 스핀 : 회전공격을 하여 적을 두번 공격합니다.
- 폭탄투척 : 폭탈을 투척함
- 독구름폭탄 : 독구름을 생성하는 폭탄을 던져서 적을 중독시킵니다.
- 곰덫 : 바닥에 곰덫을 설치해 일정시간 적을 기절시킵니다.
- 폭발덫 : 바닥에 폭발덫을 설치해 적이 밟으면 폭발시켜 적을 공격합니다.
- 단검투척 : 단검을 투척함
- 이중속박 : 쇠고랑 두개를 순서대로 발사해서 서로 당기게 만듬, 적들끼리 당기게 할 수 있고 오브젝트끼리 당기게 할 수 있음

3. 전사
- 쉴드마스터(패시브) : 방패로 방어가 가능합니다.
- 반사(패시브) : 방패로 투사체를 막았을때 확률적으로 반사합니다. 스킬레벨이 높을수록 확률이 올라감
- 강타 : 적을 강하게 칼로 내려침
- 쉴드배쉬 : 방패로 적을 밀어 기절시킴
- 소용돌이 : 360도로 돌면서 주위의 적을 공격합니다.
- 힘줄절단 : 적의 힘줄을 절단해 이동속도와 공격속도를 감소시킵니다.
- 최후의 일격 : 모든 분노를 소모해서 공격합니다. 소모한 분노만큼 데미지가 증가합니다.
- 철옹성 : 방어력이 대폭 증가하고 5초 이후에 분노가 초당 10씩 감소하여 분노가 0이 되면 효과가 사라집니다.

4.마법사
- 특수공격(마법 오브) : 특수공격을 사용할때마다 머리위에 마법구슬을 소환함 플레이어가 적을 공격할때마다 마법구슬이 자동으로 적을 공격함 최대 5개 까지 소환 가능함 
- 파이어볼 : 폭발을 일으키는 화염구를 발사함
- 불태우기 : 전방에 화염을 발사해 적을 넉백시킴
- 얼음화살 : 얼음화살을 발사해 적중한 적의 속도를 감소시킴
- 얼음보호막 : 얼음 보호막을 생성해 일정 데미지를 막아줌
- 전류구 : 천천히 앞으로 이동하는 전류구를 소환합니다.
- 정전기 : 적에게 마법 징표를 묻혀 추가데미지를 입히게 합니다.
- 과충전(OverCharge) : 일정 시간동안 마나재생량을 증가시킴 지속시간이 끝나면 일정시간동안 마나재생량이 감소함
- 블랙홀 : 블랙홀을 만들어 적을 끌어당기고 일정시간뒤에 폭발합니다. 

5. 아처
- 챠징샷(오른쪽) : 챠징하는 만큼 강해지는 화살을 발사합니다. 넉백을 발생시킵니다.
- 더블샷 : 같은 방향으로 두발의 화살을 발사합니다.
- 트리플샷 : 관통화살 세발을 세방향으로 동시에 발사함
- 늪함정 : 함정 위의 적을 느리게 만드는 함정을 바닥에 설치합니다.
- 작살화살 : 적을 서로 당기게 하는 화살 두발을 발사합니다.
- 강철화살 : 특정개수만큼 화살을 강화합니다. 화살의 데미지를 증가시키고 맞은적을 기절시킵니다.
- 독화살 : 특정개수만큼 기본공격 화살을 독화살로 바꿉니다.
- 폭발화살 : 특정개수만큼 기본공격 화살을 폭발화살로 바꿉니다.
- 백스탭샷 : 뒤로 물러서면서 화살을 발사합니다.
- 밀어내기 : 적을 뒤로 밀어냄
- 그래플링애로스 : 갈고리 화살을 발사해 빠르게 이동하거나 적을 끌어올수 있습니다.

6. 연금술사
- 재료수집(패시브) : 연금술 재료를 적의 시체나 맵의 부술수 있는 오브젝트에서 수집함
- 물약제조 : 연금술 재료를 융합하여 새로운 물약을 만들어냄
- 물약마시기 : 제조한 물약중 하나를 마심
- 물약투척 : 제조한 물약중 하나를 적에게 투척함

7. 사령술사
- 영혼수집(패시브) : 죽은 적에게서 영혼을 수집합니다. => 1레벨당 영혼 획득량 50%씩 증가
- 네크로애로우(유령미사일) : 일정량의 영혼을 소모해 마법미사일을 날림, 이 마법화살은 유도기능이 있음
- 하베스트 : 지하연옥에서 일정량의 영혼을 소환시킴 소환되는 영혼의 수와 크기는 랜덤(예 : 랜덤의 영혼해골이 땅에서 솓아나오도록함 ) 쿨타임이 김
- 컨슘 : 영혼을 모두 소모해 큰 폭발을 일으킴
- 스켈레톤 소환 : 영혼을 소모해 스켈레톤을 소환해서 전투를 지원하게함
- 스켈레톤 기사 소환 : 영혼을 소모해 스켈레톤 기사를 소환해서 전투를 지원하게함
- 스켈레톤 궁수 소환 : 영혼을 소모해 스켈레톤 궁수를 소환해서 전투를 지원하게함
- 리치 소환 : 강력한 리치를 소환해서 전투를 지원하게 함
- 죽음의 오라 : 플레이어 주변에 오라를 만들어 오라에 들어온 적들에게 데미지를 줍니다.
- 영혼 희생 : 모든 영혼을 폭발시켜 법위안에 대상에게 피해를 줍니다.
- 연옥의 족쇠 : 연옥의 사슬로 적을 속박시킵니다.
- 영혼의 불꽃 : 전방으로 화염을 발사해 적에게 데미지를 입히고 화상 디버프를 겁니다.

성향
*플레이어의 성향에 따라 신들의 축복을 받을 수 있습니다.

선성향 : 
악성향 :

성향을 얻는법 : 플레이어의 여러 행동에 따라 성향 포인트를 얻을 수 있습니다. 
1. 상대 석상을 부수기 = 여신의 석상 또는 악마의 석상을 부수어서 성향을 획득 가능
2. 기도하기 = 여신의 석상 또는 악마의 석상에 기도하여 포인트를 획득가능
3. 제물바치기 = 제물을 바쳐 악성향을 얻을 수 있습니다.
4. 공물바치기 = 공물을 바쳐 선성향을 얻을 수 있습니다.
5. 중립NPC죽이기 = 선성향은 하락하고 악성향은 증가한다.
6. NPC구출 = 갇혀있는 NPC를 구출하면 선성향이 증가하고 악성향이 하락한다.

성향으로 악세서리 얻기 : 성향이 높아지만 악세서리를 얻을 수 있습니다.
